## Tasks CLI

programa para administrar sus tareas en una base de datos usando terminal o consola. 

### Requirements

- Nodejs
- Mongodb Local Installation or Cloud

### Installation

```sh
npm install
```

```sh
npm link
```

### Commands

```
tasks-cli --version
```

```
tasks-cli --help
```

```
tasks-cli list
tasks-cli l
```

```
tasks-cli find title
```

```sh
tasks-cli save
tasks-cli s
```

```sh
tasks-cli update <id>
```

```sh
tasks-cli delete <id>
tasks-cli d <id>
```
